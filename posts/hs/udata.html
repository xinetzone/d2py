恒有数 Python 接口¶

恒有数 提供了大量金融数据相关的接口，借助此平台讲解 Python 基础知识。

支持平台：

  1. Window10 或者更高版本的系统平台

  2. Ubuntu20.04 或者更高版本

后续的教程均指代 Python 为 Python3.10。

配置 conda 环境¶

使用 conda 创建并激活环境 hs：

conda create -n hs python=3.10
conda activate hs

安装 hs 环境的 jupyter 内核（为了方便 vscode 等编辑器使用）：

conda install ipykernel
python -m ipykernel install --user --name hs

安装 Jupyter 插件¶

参考：ipython-contrib/jupyter_contrib_nbextensions

安装插件：

pip install jupyter_contrib_nbextensions

安装 javascript 和 css 文件：

jupyter contrib nbextension install --user

安装代码格式化工具：

pip install autopep8

安装恒有数的相关包¶

警告

需要配置 conda 环境 hs。

  1. 在恒有数主页 完成注册登录，可用手机号或微信进行注册登录；

  2. 使用 pip 安装需要的包（文件在 pip-env/hs_udata.txt）：

cd docs
pip install -r pip-env/hs_udata.txt

祝贺，全部的准备工作完成了！
