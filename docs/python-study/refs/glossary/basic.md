# 基本术语

```{glossary}
data
数据
    描述了事实或观察的结果，是对客观事物的逻辑归纳，是用于表示客观事物的未经加工的原始素材。

keyword
关键字
    Python 定义的一些具有特殊语义或者逻辑的符号。

代码块 
    作为一个单元来执行的一段 Python 程序文本。换言之，代码块可能由多条 {dfn}`语句` 组成。

异常
    即使语句使用了正确的语法，执行时仍可能触发错误。执行时检测到的错误称为 **异常**。

    {dfn}`异常` 是中断代码块的正常控制流程以便处理错误或其他异常条件的一种方式。异常会在错误被检测到的位置引发，它可以被当前包围代码块或是任何直接或间接发起调用发生错误的代码块的其他代码块所处理。

procedure
过程
    不返回有意义结果的函数；在 Python 中，过程的返回值为 `None`。
```
