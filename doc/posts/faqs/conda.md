# Conda 常见问题

## 修改 conda 缓存路径

只需配置文件 `~/.condarc` 即可：

```
envs_dirs:                                                                                                                                 
  - /media/pc/data/tmp/cache/conda/envs                                                                                                                              
pkgs_dirs:                                                                                                                                 
  - /media/pc/data/tmp/cache/conda/pkgs
```

## conda 解决“libstdc++.so.6: version `GLIBCXX_3.4.20‘ not found“

```bash
conda install -c anaconda libstdcxx-ng
conda install -c conda-forge gcc
```
